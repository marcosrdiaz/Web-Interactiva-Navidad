# Web Interactiva "Santa Tracker"
***Breve descripción***
Sitio web interactivo navideño que incluye registro, juegos, cartas a Papá Noel, reserva de viajes, visualización de mapa con ruta en tiempo real y reproducción de villancicos.

***Qué hace el proyecto***
Ofrece entretenimiento y herramientas para la temporada navideña: escribir cartas, reservar viajes, ver ubicación de Papá Noel en un mapa con Leaflet, escuchar villancicos y jugar juegos temáticos.

***Tecnologías usadas***
- HTML, CSS, JavaScript (con Figma para prototipado previo)

- Librería Leaflet para mapa interactivo

- Almacenamiento en LocalStorage para datos de usuario y persistencia

- Uso ético y accesible con atención a contraste, usabilidad y privacidad

***Cómo se ejecuta***
- Abrir el proyecto en un navegador web moderno

- Navegar por las secciones: registro, juegos, cartas, mapa, villancicos

- Interactuar con formularios y funcionalidades directamente en la web

***Ejemplo de salida o captura***
- Visualización del mapa con ícono móvil de Papá Noel

- Formularios para registrar usuario y enviar cartas

- Interfaz gráfica con botones, ventanas emergentes y elementos visuales animados

- Reproductor de villancicos con lista reproducible y controles
